# こぶたしゃしんかん 🌸

フォートナイトスタイルの和装前撮り・フォトウェディングサイト

## 🎮 デザイン

- **スタイル**: Fortnite風ゲーミングUI
- **カラーパレット**: パープル、イエロー、ブルーのネオンカラー
- **エフェクト**: グローエフェクト、グラデーション、アニメーション

## 🚀 ローカル実行

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開く

## 📦 ファイル構成

- `index.html` - メインHTML
- `style.css` - フォートナイト風スタイルシート
- `script.js` - インタラクティブ機能
- `assets/` - 画像ファイル
- `robots.txt` - SEO設定
- `sitemap.xml` - サイトマップ

## 🔍 SEO対策

- OGP/Twitter Cardタグ
- 構造化データ（JSON-LD）
- sitemap.xml/robots.txt
- 最適化されたalt属性
